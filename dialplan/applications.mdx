## What are Applications?

Applications are the actions that Asterisk executes for each priority in an extension. They perform the actual work of call handling: dialing, playing audio, sending to voicemail, routing, etc.

```conf
exten => extension,priority,Application(arguments)
```

<Info>
Every dialplan action is an application. From ringing a phone to playing music, applications are the verbs of the dialplan language.
</Info>

## Common Applications

Here are the most frequently used applications from the Asterisk tutorial:

### Dial() - Place Calls

The most important application - places calls to devices or channels.

**Syntax:** `Dial(Technology/Resource,timeout,options)`

#### Basic Examples

```conf extensions.conf
[empleado]
; Ring a single SIP device for 15 seconds
exten => 4000,1,NoOp(Llamando a Javier)
same => n,Dial(SIP/javier,15)
same => n,VoiceMail(${EXTEN})
same => n,Hangup()
```

#### Group Calling

Ring multiple devices simultaneously (first to answer gets the call):

```conf extensions.conf
[entrantes]
; Ring javier AND belen for 60 seconds
exten => 4040,1,NoOp(Llamando al grupo 'empleados')
same => n,Dial(SIP/javier&SIP/belen,60)
same => n,Hangup()

[llamar-grupo]
; Reception group - 4 phones ring
exten => 100,1,Dial(SIP/101&SIP/102&SIP/103&SIP/110,60)
same => n,Hangup()

; Management group
exten => 200,1,Dial(SIP/201&SIP/202&SIP/203,60)
same => n,Hangup()

; Doctors group
exten => 300,1,Dial(SIP/301&SIP/302&SIP/303,60)
same => n,Hangup()
```

<Tip>
Use `&` to ring multiple devices at once. The ampersand means "AND" - all devices ring simultaneously.
</Tip>

#### Dial Options

Common options passed as the third parameter:

```conf
Dial(SIP/javier,15,tTr)  ; t=allow callee transfer, T=allow caller transfer, r=ring tone
```

**Common Options:**
- `t` - Allow called party to transfer
- `T` - Allow calling party to transfer
- `r` - Generate ringing tone for caller
- `m` - Provide music on hold
- `p` - Call screening/privacy mode

### VoiceMail() - Leave Messages

Send caller to voicemail.

**Syntax:** `VoiceMail(mailbox,options)`

```conf extensions.conf
[empleado]
exten => 4000,1,NoOp(Llamando a Javier)
same => n,Dial(SIP/javier,15)
same => n,VoiceMail(${EXTEN})  ; Send to mailbox 4000
same => n,Hangup()

exten => 4001,1,NoOp(Llamando a Belen)
same => n,Dial(SIP/belen,15)
same => n,VoiceMail(${EXTEN})  ; Send to mailbox 4001
same => n,Hangup()
```

**Options:**
- `u` - Play "unavailable" greeting
- `b` - Play "busy" greeting
- `s` - Skip greeting, go straight to beep

```conf
exten => stdexten-NOANSWER,1,VoiceMail(${mbx},u)  ; Unavailable message
exten => stdexten-BUSY,1,VoiceMail(${mbx},b)      ; Busy message
```

### VoiceMailMain() - Check Voicemail

Access voicemail inbox.

**Syntax:** `VoiceMailMain(mailbox)`

```conf extensions.conf
[empleado]
; Direct voicemail access
exten => *4000,1,VoiceMailMain(4000)
same => n,Hangup()

exten => *4001,1,VoiceMailMain(4001)
same => n,Hangup()
```

#### Interactive Voicemail Access

From the call center example with PIN authentication:

```conf extensions.conf
[buzon-voz]
exten => 1,1,Festival(Para acceder a su buzon de voz)
same => n(pin),Festival(Marque su numero de identificacion privado)
same => n,Read(pin,,4,3,5)
same => n,Set(idpin=${ODBC_pin(${llamante})})
same => n,GotoIf($["${pin}" = "${idpin}"]?2,1)
same => n,Festival(Numero de identificacion erroneo)
same => n,Goto(1,pin)
same => n,Hangup()

exten => 2,1,Festival(Marque 1 para continuar)
same => n,VoiceMailMain(${llamante})
same => n,Hangup()
```

### Playback() - Play Audio

Play a sound file to the caller.

**Syntax:** `Playback(filename,options)`

```conf extensions.conf
[empleado]
; Play music from custom directory
exten => 100,1,NoOp(Centralita de distribucion de llamadas)
same => n,Playback(silence/1)           ; 1 second silence
same => n,Playback(mimusica/primavera)  ; Custom music file
same => n,Hangup()
```

<Info>
Audio files are stored in `/var/lib/asterisk/sounds/` (plus language directory, e.g., `/es/` for Spanish). Asterisk automatically adds the file extension.
</Info>

**File formats:** `.wav`, `.gsm`, `.ulaw`, `.alaw`, `.sln16`

```conf
; Standard prompts
exten => demo,1,Playback(demo-congrats)    ; Plays demo-congrats.gsm
same => n,Playback(demo-thanks)
```

### BackGround() - Play Audio with DTMF

Like Playback(), but allows caller to interrupt by pressing digits.

```conf extensions.conf
[demo]
exten => s,1,Answer()
same => n,Set(TIMEOUT(digit)=5)
same => n(restart),BackGround(demo-congrats)
same => n(instruct),BackGround(demo-instruct)
same => n,WaitExten()  ; Wait for digit press
```

### Festival() - Text to Speech

Convert text to speech using Festival TTS.

**Syntax:** `Festival(text)`

```conf extensions.conf
[exterior]
exten => 757112233,1,GotoIfTime(8:00-20:00,mon-fri,*,*?abierto,1,1)
same => n,Festival(En estos momentos no hay nadie que te pueda atender)
same => n,Festival(Nuestro horario es de 8 a 20 horas de lunes a viernes)
same => n,Hangup()

[abierto]
exten => 1,1,Answer()
same => n,Festival(Bienvenido al centro de salud)
same => n(loop),Festival(Presione)
same => n,SayNumber(1)
same => n,Festival(si es un usuario registrado o)
same => n,SayNumber(2)
same => n,Festival(en caso contrario)
```

#### Festival with Variables

```conf extensions.conf
[recepcion]
exten => 1,1,Set(CHANNEL(musicclass)=recepcion)
same => n,Answer()
same => n,Festival(Hola ${nombre})  ; Dynamic greeting
same => n(reg),Festival(Por favor espere a que le atiendan)

[llamar]
exten => _XXX,1,Festival(Llamando a ${nombre})
same => n,Dial(SIP/${extension},15)

[direccion]
exten => _X00,1,Festival(Llamada a grupo ${EXTEN})
same => n,Goto(llamar-grupo,${EXTEN},1)
```

<Tip>
Festival requires installation and configuration. See the tutorial section 4.6 for setup instructions.
</Tip>

### Queue() - Call Queues

Place caller in a queue to be answered by available agents.

**Syntax:** `Queue(queuename,options,URL,announceoverride,timeout)`

```conf extensions.conf
[recepcion]
; Registered users queue
exten => 1,1,Set(CHANNEL(musicclass)=recepcion)
same => n,Answer()
same => n,Festival(Hola ${nombre})
same => n(reg),Festival(Por favor espere a que le atiendan)
same => n,Queue(registrados,,,,60)  ; 60 second timeout
same => n,Goto(1,reg)  ; Loop if no answer
same => n,Hangup()

; Unregistered users queue
exten => 2,1,Set(CHANNEL(musicclass)=recepcion)
same => n,Answer()
same => n(noreg),Festival(Por favor espere a que le atiendan)
same => n,Queue(noregistrados,,,,60)
same => n,Goto(2,noreg)
same => n,Hangup()
```

<Info>
Queues must be defined in `/etc/asterisk/queues.conf` before use. The Queue() app places the caller in that queue.
</Info>

### Goto() - Jump to Extension

Jump to a different context, extension, or priority.

**Syntax:** `Goto([[context,]extension,]priority)`

```conf extensions.conf
; Jump to different context
exten => _995554XXX,1,Set(extension=${EXTEN:5})
same => n,NoOp(Llamando a la extension ${extension})
same => n,Goto(llamar,${extension},1)  ; Jump to [llamar] context

; Jump to labeled priority
exten => 1,1,Answer()
same => n(loop),Festival(Presione 1 o 2)
same => n,Read(opcion,,1,3)
same => n,GotoIf($["${opcion}" = "1"]?10,1)
same => n,Festival(Opcion incorrecta)
same => n,Goto(1,loop)  ; Jump back to (loop) label
```

**Forms:**
```conf
Goto(priority)                    ; Same context and extension
Goto(extension,priority)          ; Same context
Goto(context,extension,priority)  ; Different context
```

### GotoIf() - Conditional Jump

Jump based on a condition.

**Syntax:** `GotoIf(condition?destination_if_true:destination_if_false)`

#### Basic Conditional

```conf extensions.conf
[abierto]
exten => 1,1,Read(opcion,,1,3)
same => n,GotoIf($["${opcion}" = "1"]?10,1)  ; If opcion is 1, goto extension 10
same => n,GotoIf($["${opcion}" = "2"]?20,1)  ; If opcion is 2, goto extension 20
same => n,Festival(Opcion incorrecta)
```

#### Time-Based Routing

```conf extensions.conf
[exterior]
exten => 757112233,1,GotoIfTime(8:00-20:00,mon-fri,*,*?abierto,1,1)
same => n,Festival(En estos momentos no hay nadie que te pueda atender)
```

#### Database Lookup

```conf extensions.conf
[abierto]
exten => 10,1(id),Festival(Por favor marque su numero de identificacion)
same => n,Read(id,,4,3,5)
same => n,Set(nombre=${ODBC_usuario(${id})})
same => n,GotoIf($["${nombre}" != ""]?recepcion,1,1)
same => n,Festival(Usario no encontrado)
same => n,Goto(10,id)

[recepcion]
exten => _XXX,1,Set(extension=${EXTEN})
same => n,Set(nombre=${ODBC_extension(${extension})})
same => n,GotoIf($["${nombre}" != ""]?llamar,${extension},1)
same => n,Festival(La extension ${extension} no se encuentra disponible)
```

#### Permission Check

```conf extensions.conf
[doctores]
exten => _*8.,1,Set(extension=${EXTEN:2})
same => n,Set(pref=${extension:0:1})
same => n,GotoIf($["${pref}" = "3"]?llamar,8,1)  ; Only capture group 3
same => n,Festival(No tiene permiso para capturar una llamada a la extension ${extension})
same => n,Hangup()
```

### Pickup() - Call Pickup

Pickup a ringing call at another extension.

**Syntax:** `Pickup(extension[@context])`

```conf extensions.conf
[empleado]
; Pickup within same context
exten => _*8.,1,NoOp(Capturando llamada a la extension ${EXTEN:2})
same => n,Pickup(${EXTEN:2})
same => n,Hangup()

[direccion]
; Pickup from different context
exten => _*8.,1,NoOp(Capturando llamada a la extension ${EXTEN:2})
same => n,Pickup(${EXTEN:2}@empleado)
same => n,Hangup()

[llamar]
; Generic pickup
exten => 8,1,Festival(Capturando llamada a la extension ${extension})
same => n,Set(equipo=${ODBC_equipo(${extension})})
same => n,Pickup(${extension}@${equipo})  ; Dynamic context
same => n,Hangup()
```

<Info>
Call pickup respects the `callgroup` and `pickupgroup` settings in `sip.conf`. Users can only pickup calls from extensions in their authorized pickup groups.
</Info>

### MessageSend() - Send SIP Messages

Send SIP MESSAGE (text message) to a destination.

**Syntax:** `MessageSend(destination,from)`

```conf extensions.conf
[mensajes]
exten => _X.,1,Set(dest=${EXTEN})
same => n,Set(remitente=${CUT(MESSAGE(from),<,2)})
same => n,Set(remitente=${CUT(remitente,@,1)})
same => n,Set(remitente=${CUT(remitente,:,2)})
same => n,Set(texto=${MESSAGE(body)})
same => n,Set(MESSAGE(body)=${remitente}: ${texto})
same => n,MessageSend(sip:${dest}, Centro de Mensajes(CdM))
same => n,Noop(Estado del mensaje ${MESSAGE_SEND_STATUS})
same => n,GotoIf($["${MESSAGE_SEND_STATUS}" != "SUCCESS"]?fallo,s,1)
same => n,Hangup()

[fallo]
exten => s,1,Set(MESSAGE(body)=CdM: El mensaje "${texto}" para ${dest} no ha sido enviado)
same => n,MessageSend(${remit},Centro de Mensajes(CdM))
same => n,Hangup()
```

### NoOp() - No Operation

Does nothing except log to console - useful for debugging.

**Syntax:** `NoOp(message)`

```conf extensions.conf
exten => 4000,1,NoOp(Llamando a Javier)
exten => 4001,1,NoOp(Llamando a Belen)
exten => _X.,1,NoOp(Mensaje de ${MESSAGE(from)})

exten => 101,1,NoOp(>>>>>>>>CALLERID = ${CALLERID(name)}-------${CALLERID(number)})

[mensajes]
exten => _X.,1,NoOp(Mensaje de ${MESSAGE(from)})
same => n,NoOp(Mensaje para ${MESSAGE(to)})
same => n,NoOp(Cuerpo del mensaje: ${MESSAGE(body)})
```

<Tip>
Use NoOp() liberally during development to trace dialplan execution in the Asterisk console.
</Tip>

### Hangup() - End Call

Terminate the current call.

```conf extensions.conf
exten => 4000,1,Dial(SIP/javier,15)
same => n,VoiceMail(${EXTEN})
same => n,Hangup()  ; Always end with Hangup()
```

### Answer() - Answer Channel

Answer the incoming call.

```conf extensions.conf
[abierto]
exten => 1,1,Answer()  ; Answer before playing prompts
same => n,Festival(Bienvenido al centro de salud)

[recepcion]
exten => 1,1,Set(CHANNEL(musicclass)=recepcion)
same => n,Answer()
same => n,Festival(Hola ${nombre})
```

### Read() - Get User Input

Read DTMF digits from the caller.

**Syntax:** `Read(variable,filename,maxdigits,option,attempts,timeout)`

```conf extensions.conf
[abierto]
; Read 1 digit, 3 attempts, timeout 5 seconds
exten => 1,1(loop),Festival(Presione 1 o 2)
same => n,Read(opcion,,1,3,5)
same => n,GotoIf($["${opcion}" = "1"]?10,1)

; Read 4-digit ID
exten => 10,1(id),Festival(Marque su numero de identificacion)
same => n,Read(id,,4,3,5)
same => n,Set(nombre=${ODBC_usuario(${id})})

[buzon-voz]
; Read PIN
exten => 1,1(pin),Festival(Marque su numero de identificacion privado)
same => n,Read(pin,,4,3,5)
same => n,Set(idpin=${ODBC_pin(${llamante})})
same => n,GotoIf($["${pin}" = "${idpin}"]?2,1)
```

### SayNumber() - Speak Number

Speak a number to the caller.

```conf extensions.conf
[abierto]
exten => 1,1(loop),Festival(Presione)
same => n,SayNumber(1)
same => n,Festival(si es un usuario registrado o)
same => n,SayNumber(2)
same => n,Festival(en caso contrario)
```

## Application Reference

View all available applications:

```bash
asterisk -rx "core show applications"
```

```bash
asterisk -rx "core show application Dial"
asterisk -rx "core show application Queue"
```

From Asterisk CLI:

```
CLI> core show applications
CLI> core show application VoiceMail
```
