title: 'Quickstart Guide'
description: 'Get Asterisk up and running with your first test call in minutes'
---

Get from zero to making calls between two test users in just a few steps. This guide uses Debian 9+ and Asterisk 15.7.1.

<Steps>

<Step title="Update system and install dependencies">
First, ensure your system is up to date and install the required build tools:

```bash
sudo apt update && sudo apt upgrade
sudo apt install wget build-essential subversion
```
</Step>

<Step title="Download and extract Asterisk">
Download Asterisk 15.7.1 and extract it to `/usr/src/`:

```bash
cd /usr/src/
sudo wget http://downloads.asterisk.org/pub/telephony/asterisk/releases/asterisk-15.7.1.tar.gz
sudo tar zxf asterisk-15.7.1.tar.gz
```
</Step>

<Step title="Install prerequisites and prepare compilation">
Navigate to the Asterisk directory and install dependencies:

```bash
cd asterisk-15.7.1/
sudo contrib/scripts/get_mp3_source.sh
sudo contrib/scripts/install_prereq install
```

When prompted, select your country code. This process will install necessary dependencies including development libraries.

Prepare the source code for compilation:

```bash
sudo ./configure
```

You should see a confirmation message that Asterisk has been configured successfully.
</Step>

<Step title="Select modules, compile, and install">
Select the modules to compile (you can use the defaults):

```bash
sudo make menuselect
```

Compile Asterisk (this may take several minutes):

```bash
sudo make -j2
```

Install Asterisk and configuration files:

```bash
sudo make install
sudo make samples
sudo make config
sudo ldconfig
```
</Step>

<Step title="Create asterisk user">
Create a dedicated user for Asterisk to run securely:

```bash
sudo adduser --system --group --home /var/lib/asterisk --no-create-home --gecos "Asterisk PBX" asterisk
```

Configure Asterisk to run as this user:

```bash
sudo vim /etc/default/asterisk
```

Uncomment these lines:

```
AST_USER="asterisk"
AST_GROUP="asterisk"
```

Add the asterisk user to required groups and set permissions:

```bash
sudo usermod -a -G dialout,audio asterisk
sudo chown -R asterisk: /var/{lib,log,run,spool}/asterisk /usr/lib/asterisk /etc/asterisk
sudo chmod -R 750 /var/{lib,log,run,spool}/asterisk /usr/lib/asterisk /etc/asterisk
```

Start Asterisk and enable it to start on boot:

```bash
sudo /etc/init.d/asterisk start
sudo systemctl enable asterisk
```
</Step>

<Step title="Configure two test users">
Now let's configure two SIP users (javier and belen) to test calls between them.

Edit the SIP configuration file:

```bash
sudo vim /etc/asterisk/sip.conf
```

Add the following configuration:

<CodeGroup>
```ini sip.conf
[general]
port=5060
directmedia=no
language=es
context=public

[javier]
type=friend
secret=12345678
context=empleado
host=dynamic
canreinvite=no
nat=force_report

[belen]
type=friend
secret=12345678
context=empleado
host=dynamic
canreinvite=no
nat=force_rport,comedia
```
</CodeGroup>

<Note>
- **port=5060**: Standard SIP port for connections
- **directmedia=no**: Routes RTP media through Asterisk (better compatibility)
- **type=friend**: Allows sending and receiving calls
- **secret**: Password for authentication
- **context**: Dialplan context for this user
- **host=dynamic**: Allows dynamic registration
</Note>

Now configure the dialplan to route calls:

```bash
sudo vim /etc/asterisk/extensions.conf
```

Add this context:

<CodeGroup>
```ini extensions.conf
[empleado]
exten => 4000,1,Dial(SIP/javier)
exten => 4001,1,Dial(SIP/belen)
```
</CodeGroup>

This assigns extension 4000 to javier and 4001 to belen.

Reload the configuration:

```bash
sudo asterisk -r
```

In the Asterisk CLI:

```
CLI> dialplan reload
CLI> sip reload
```
</Step>

<Step title="Make your first test call">
Install a SIP softphone on your devices. Popular options include:

- **Zoiper** (Windows, macOS, Linux, iOS, Android)
- **Linphone** (Windows, macOS, Linux, iOS, Android)
- **Blink** (Windows, macOS, Linux)

Configure the first softphone with these settings:

<CodeGroup>
```yaml Javier's Settings
Username: javier
Password: 12345678
Domain: [Your Asterisk server IP]
Port: 5060
```

```yaml Belen's Settings
Username: belen
Password: 12345678
Domain: [Your Asterisk server IP]
Port: 5060
```
</CodeGroup>

Once both clients register successfully, dial **4001** from javier's phone to call belen, or **4000** from belen's phone to call javier.

<Note>
Check registration status in the Asterisk CLI:
```
CLI> sip show peers
```

You should see both users with status "OK" and their IP addresses.
</Note>

Monitor the call in real-time from the Asterisk CLI:

```bash
sudo asterisk -rvvvvvvc
```

The verbose mode will show the complete call flow including registration, dialing, ringing, and connection.

Congratulations! You've successfully set up Asterisk and made your first call.
</Step>

</Steps>

## Troubleshooting

If you encounter issues:

**Can't connect to Asterisk CLI**
```bash
# Check if Asterisk is running
sudo /etc/init.d/asterisk status

# Try restarting
sudo /etc/init.d/asterisk restart
```

**Softphone won't register**
- Verify the server IP address is correct
- Check firewall allows UDP port 5060
- Confirm credentials match sip.conf
- Check Asterisk logs: `tail -f /var/log/asterisk/messages`

**No audio during calls**
- Verify firewall allows RTP ports (typically 10000-20000 UDP)
- Check `directmedia=no` is set in sip.conf
- Confirm NAT settings are correct for your network
